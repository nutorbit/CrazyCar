# Crazy Car with Reinforcement Learning

## Setup Environment

```shell
$ pip install -r requirements.txt
```

## Running the code

**training a model.**

```shell
# SAC
$ python -m sac_torch.sac

# TD3
$ python -m td3_torch.td3
```

**testing a model.**

```shell
# SAC
$ python -m sac_torch.test path

# TD3
$ python -m td3_torch.test path
```
